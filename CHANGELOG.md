## 0.0.1

* Initial release of flutter_live_motion plugin.
* Support for generating iOS Live Photos with proper metadata pairing.
* Support for generating Android Motion Photos with XMP metadata.
* Compatible with multiple Android vendors:
  - Google Pixel (Google Photos)
  - OPPO/Realme (ColorOS/RealmeUI)
  - Xiaomi (MIUI/HyperOS)
* Smart XMP merging that preserves original image metadata (HDR GainMap, etc.).
* Automatic saving to device photo library (iOS) or cache directory (Android).
* Example app demonstrating plugin usage with image and video pickers.
